## ⚡ Overview

**MGVerse** is not a standard web portfolio. It is engineered as a highly immersive, futuristic "Command Center." The UI blends neo-glassmorphism, cinematic noise overlays, and dynamic interactive elements to create a portfolio that feels alive, premium, and inherently technical.

## 🛠 Tech Stack

- **Structure:** HTML5 (Semantic & Accessible)
- **Styling:** Tailwind CSS (Utility-first framework)
- **Interactions & Visuals:** Vanilla JavaScript & CSS3 (For complex math, SVG injection, and Canvas rendering)
- **Icons:** Inline SVGs & Lucide

## 🚀 Key Features

*   **Interactive Particle Canvas:** A reactive, node-based background network running natively in JavaScript.
*   **Dynamic SVG Laser Tracing:** Instead of basic borders, JS dynamically calculates and injects glowing SVG borders (`stroke-dasharray`) that trace the physical edges of UI cards on hover.
*   **Asymmetrical Bento Grids:** Custom, highly dense masonry and bento-box layouts to prevent empty space and maximize data density.
*   **Cybernetic Timeline:** A non-traditional staggered zigzag career timeline.
*   **Cinematic Textures:** Global noise overlays (`film-grain`) and deep blurred parallax midgrounds to mimic Awwwards-winning depth.
